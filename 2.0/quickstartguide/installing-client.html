    Installing the client package

    For the simplest installation experience, install the client package. This package can be installed on Linux, Mac OSX, and Windows. It contains everything that you need to run the DITA-OT except for Java.

    Before you begin

    • Ensure that you have Java JRE or JDK, version 7 or later installed.
    • Ensure that you have HTML Help Workshop installed, if you want to generate HTML Help.

    Procedure

    1. Download the client package from SourceForge.
      Operating system File name
      Linux or Mac OSX DITA-OT2.0_client_bin.tar.gz
      Windows DITA-OT2.0_client_bin.zip
    2. Extract the contents of the package to the directory where you want to install the DITA-OT.
    3. Optional: Add absolute path for bin directory to PATH system variable.
      This defines the necessary environment variable to run the dita command from the command-line.
